# Raw sources (clips)

Drop external articles, meeting notes, and research clips here for **maintain-wiki ingest**.

- Files in this folder are **immutable** once dropped — the LLM reads but does not edit them.
- Images from web clips go in `assets/`.
- After adding a file, run **maintain-wiki ingest** (not sync).

Sync reads operational repo sources only; ingest handles new clips in this folder.
